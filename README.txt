Football Club Management System
==============================

This is a complete football club management system built with PHP and MySQL.

Features:
- Player management (add, edit, delete, view)
- Staff management (add, edit, delete, view)
- Team management (add, edit, delete, view)
- Match scheduling (add, edit, delete, view)
- User authentication

Installation:
1. Create a MySQL database named 'football_club'
2. Import the SQL schema (provided separately)
3. Update the database credentials in config.php
4. Upload all files to your web server
5. Access the system via your web browser

Default Admin Credentials:

Password: password123

Note: This is a basic system for demonstration purposes. In a production environment, you should:
- Implement proper password hashing

- Implement input validation
- Add more features as needed
